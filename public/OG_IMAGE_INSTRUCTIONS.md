# OG Image Setup Instructions

## Current Status
The SEO component is currently using `/favicon.svg` as a fallback for the Open Graph image.

## Recommended OG Image Specifications

### Dimensions
- **Width:** 1200px
- **Height:** 630px
- **Aspect Ratio:** 1.91:1

### Format
- **Preferred:** JPG or PNG
- **File Size:** Under 1MB (optimized for web)

### Content Suggestions
The OG image should include:
1. RoundDigital logo
2. Tagline: "Enterprise IT Solutions, AI Development & Cybersecurity"
3. Visual elements that represent technology/AI/cloud
4. Brand colors (#e14242 red, #191a23 dark)

### File Location
Once created, place the image at:
```
/public/images/og-image.jpg
```

The SEO component will automatically use it.

### Tools for Creating OG Images
- Canva (OG Image template: 1200x630)
- Figma
- Adobe Photoshop
- Online tools like og-image.vercel.app

### Testing
After creating the image, test it using:
- Facebook Sharing Debugger: https://developers.facebook.com/tools/debug/
- Twitter Card Validator: https://cards-dev.twitter.com/validator
- LinkedIn Post Inspector: https://www.linkedin.com/post-inspector/
