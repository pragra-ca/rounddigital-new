## 🎯 After Pushing to GitHub

### 1. Deploy Strapi to Strapi Cloud
- Go to: https://cloud.strapi.io/
- Connect your GitHub repo (strapi-backend folder)
- Add environment variables
- Deploy
- **Save your Strapi URL**

### 2. Deploy Frontend to Vercel
- Go to: https://vercel.com/
- Connect your GitHub repo (frontend folder)
- Add environment variables:
  ```
  NEXT_PUBLIC_STRAPI_URL=https://your-strapi-url.com
  EMAIL_USER=your-gmail@gmail.com
  EMAIL_PASS=your-app-password
  EMAIL_RECEIVER=info@rounddigital.co
  ```
- Deploy

### 3. (Optional) Deploy Backend
- Use Railway or Render
- Only if you need custom API endpoints
